Food Advisor - Database Management Project
Description

In this project, we build a PHP based web application with a PostgreSQL database. This web application runs on the local server. This application is a Food Blog where users can share their food experiences from different restaurants. Based on their rating, users' home page & restaurants' home page shows most popular food items. All the data are saved & updated in local database.

Features
  • User can write their food experiences with images.
  • Restaurants & Users have their own profile.
  • Every restaurants and their all food items have own rating based on users' average rating.
Online Judge - Java Based FTP Project
Description

Here, we build an online programming code compiler & judging desktop application. We use Java & JavaFX. Here a user uploads their C or Java code file to the local server for a specific programming problem and the server gives a verdict (eg. Accepted / Run time error / Syntax error) based on their code.

Features
  • User can upload their C or Java code.
  • Server receives users' code files, compiles them and gives a verdict.
Shooting Game - Micro Controller Project
Description

Here, we build a microcontroller circuit with two Bluetooth sensors where two players can play shooting game on the led matrix board with the phone apps.

Features
  • Players can shoot their opponent simultaneously using phone apps connecting with circuit by bluetooth.

Description
